# AppNest 

this is a web application of to install or explore application in browser. but in this you will get just the ui and authentication

## Tech Stack

- **Frontend Framework**: React
- **Routing**: React-router
- **UI Components**: DaisyUi with Tailwind CSS
- **Slide Show**: Swiper
- **Popup massage**: SweetAlert2 
- **Styling**: Tailwind CSS v4 with custom design tokens
- **icons**: Lucid icon
- **Authentication**: firebase
